============
Requirements
============

To access all features in QuantUS from any major operating system, all that's required is Python3.9+. Conda is preferred to manage Python environments
but there's also support for the pip-based `virtualenv`_ for legacy users.

* `Python`_
* `Conda`_ (preferred)

.. _virtualenv: https://pypi.org/project/virtualenv/
.. _Python: https://www.python.org/downloads/
.. _Conda: https://conda.io/projects/conda/en/latest/user-guide/install/index.html
